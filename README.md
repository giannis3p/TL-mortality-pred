## This repository contains the code accompanying the manuscript titled:
# Prospective Multicenter Validation of Machine Learning Models for Mortality Prediction in Adult  Critically Ill Patients using Transfer Learning
The study evaluates domain adaptation (DA) and inductive transfer learning (ITL) approaches for improving generalizability of mortality prediction models across multiple ICUs.

- <code>Figures/</code>: Figures used in the manuscript.
- <code>DA models/</code>: Scripts for Domain Adaptation models.
- <code>ITL models/</code>: Scripts for Inductive Transfer Learning models.

# Installation

Python 3.12.10 and pip install -r requirements.txt

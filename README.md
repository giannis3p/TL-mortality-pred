# Installation

Python 3.12.10 and the following are required to run this code:

pip install torch==2.9.1 <br /> 
pip install lightgbm==4.6.0 <br /> 
pip install xgboost==3.1.2 <br /> 
pip install catboost==1.2.8 <br /> 
pip install numpy==2.3.5 <br /> 
pip install scikit-learn==1.8.0 <br /> 
pip install optuna==4.6.0 <br />
